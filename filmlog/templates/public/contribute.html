Contributing

If you like The Film Log and are intersted in helping out, there are a number of ways to do so, such as:

Making a Monetary Donation

This helps cover the monthly hosting fees for the public website, keeps the website ad-free and also helps cover my time to work on new features. Occasionally I also need to purchase materials to help test or improve a new or existing feature and this would help here as well.

The important takeaway is I want the film log to be free. The codebase itself is open source, though maintaining the public website does have recurring costs. The public website avoids folks needing to run their own version on theier own server (be it on a Raspberry Pi or on a cloud server).

Donate Link Goes Here

Buying a Print

Instead of making a straight up donation, you can also buy one of my prints, which are available here. If you want to make sure the proceeds go towards The Film Log, be sure to mention that in the order comments.

Log a Bug / Feature

See something on the website which is busted? Consider logging a bug or putting in a feature request, which you can do by going to the GitLab project, here.

Contributing To Development

If you are a designer, developer, etc. you can contribute to The Film Log project directly. The code is all open source (GPLv3) and is available on GitLab. The core technologies used are Python (Flask), MariaDB, JavaScript/jQuery, and CSS/HTML. A full mobile app is planned though not currently being developed.
